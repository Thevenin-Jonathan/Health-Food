Voici un processus complet pour publier une mise à jour :

Mettez à jour le numéro de version dans app_info.py
Compilez votre application avec PyInstaller
Créez l'installateur avec Inno Setup
Créez une nouvelle GitHub Release et téléversez l'installateur
Mettez à jour update_info.json avec les nouvelles informations
Publiez les changements sur GitHub
